# Notes

## What is Backend Development?

Backend development is responsible for handling business logic,
processing requests, managing databases,
authentication, and serving data to frontend applications.

---

## Django 2005

- Full-stack framework
- Built-in Authentication
- ORM
- Admin Panel

Advantages
- Secure
- Scalable
- Batteries Included

Disadvantages
- Heavy for small projects

--- 

## Flask 2010

- Lightweight
- Flexible
- Minimal

Advantages
- Easy to learn
- Good for small projects

Disadvantages
- Extra libraries needed

---

## FastAPI 2018 

- Modern framework
- Very Fast
- Automatic API Documentation
- Async Support

Advantages
- High Performance
- Easy API Development

Disadvantages
- Smaller ecosystem than Django

---

## Conclusion

Choose Django for large applications.

Choose Flask for small projects.

Choose FastAPI for APIs and AI/ML projects.
